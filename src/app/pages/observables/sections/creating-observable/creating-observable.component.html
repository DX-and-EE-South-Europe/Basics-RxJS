To create a new Observable, you can create a new Observable instance. The Observabale constructor takes a suscriber function as an argument.

The suscriber function allow us to use the methods:

  • next: emit a value like a number, string, object....
  • error: emit an error.
  • complete: complete the observable. It stops emitting more values.

It is recomended to wrap with a try/catch to throw an error if you catch an error or exception.

Also, when unsuscribing from the observable, we use the return. Really usefull to complete the observable (prevent the observable to continue emiting values), clean intervals... (See Disposing) .
