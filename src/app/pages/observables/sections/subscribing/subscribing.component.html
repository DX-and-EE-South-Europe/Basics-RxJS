Once you have an Observable, to start receiving the values emitted by the observable, you need to use the suscribe method. It is the way to start the execution of the observable.

obs$.suscribe()

When you suscribe to an Observable and later you suscribe to it again, it may not emit the same values as the previous. This is because they are independent.

If you want to receive the same emitted value, you need to multicast the original observable. There are some options:

  • Multicasted Observables.
  • Some operators: Like share.
